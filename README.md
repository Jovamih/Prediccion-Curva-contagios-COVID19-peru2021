# Prediccion de la Curva de contagios por el COVID19 en peru para el año 2021 usando Deep Learning
Repositorio con la prediccion de la curva de contagios en Peru para el el año 2021 con redes neuronales

![Curva de contagios obtenida](https://i.ibb.co/jy0dHK5/descarga.png)
* El proyecto busca estimar el  numero de contagios del covid usando la data del MINSA https://www.datosabiertos.gob.pe/dataset/casos-positivos-por-covid-19-ministerio-de-salud-minsa
* Los datos utilizados tienen sus inicios en marzo del 2020 hasta el 06 de febrero del 2021
* El autor no se responsabiliza del uso que se pueda dar a los datos predecidos debido a que se bassa en estimaciones
### Herramientas utilizadas
